# 변경 이력 (Changelog)

## v19 (현재) — 2026-05-08

### 🐰 귀요미 탭 (NEW)
- **[NEW]** 가상 토끼 (귀요미) — 클릭 시 20개 응원 메시지 랜덤 출력 + 하트 효과
- **[NEW]** 상태 시스템: 배고픔/청결도/에너지 (시간 경과로 감소)
- **[NEW]** 액션: 당근 주기 (1시간 3번, 20분 쿨다운), 목욕, 재우기
- **[NEW]** Q&A 챗봇: 8개 빠른 질문 버튼 (직접 답변 lookup, keyword 충돌 방지)
- **[NEW]** 메모장 (자유 작성·삭제, localStorage 저장)

### 입력/파싱 강화
- **[NEW]** 시간 키워드: `오전`(10-12시), `오후`(13-17시), `출근`(10시)
- **[NEW]** 언어 코드 별칭: `MA`/`ML`/`D` → I (인도네시아), `L` → T (태국/라오스)
- **[NEW]** Multi-stop lock 자유 파싱: `한승미 아트라인 출근 → 니어 1시까지 → 브라운 3시까지` (순서 자유)

### Engine 개선
- **[FIX]** allDay 강제 — 시간 미입력 demand 시 통역사는 전일 근무, 다른 지점 이동 차단
  - 조기 마감 + 이동시간 여유 시에만 예외 허용
  - allDay demand 우선 처리하여 lock-in
- **[FIX]** 입력에 없는 병원 lock 무시 + 경고 (`고정배정 무시: ...`)
- **[FIX]** `s.fixedBase` 도 active 병원에만 적용 (한미영 명동 잘못 노출 버그 fix)
- **[FIX]** 중복 배정 방지 강화 — `isLK` bypass 시 `cur===d.hospId` 또는 multi-stop만 허용
- **[FIX]** 통역사 충분히 배정됐는데 "배정 불가" 경고 뜨던 버그 수정 (defensive 재계산)

### Multi-stop Output
- **[NEW]** 출발지에만 chain 표시: `한승미 (10:00 · → 니어 1시까지 → 브라운 3시까지 이동)`
- **[NEW]** 후속 지점 자동 생략 (텍스트 결과 포함, 카톡 복사 최적)

### 알바 규칙 변경
- **[CHANGE]** 알바 주말 한정 규칙 폐지 — 각자 `days` 배열대로 출근 가능
- **[CHANGE]** 자리나 days: [5,6] → [1,4,5] (화/금/토)
- **[CHANGE]** 카잉툰 days: [0,5,6] → [0,4,5,6] (월/금/토/일)

### 인프라
- **[NEW]** GitHub 배포: https://github.com/xungmiluv1-hash/seoul-schedule
